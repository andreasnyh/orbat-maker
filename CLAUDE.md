# ORBAT Maker

## Design Context

**Brand**: Tactical, precise, functional. A milsim planning tool, not a game.

**Emotional goal**: Confidence and control — predictable, deliberate, no surprises.

**Design principles**:
1. **Function dictates form** — every visual choice serves scanning, grouping, or interaction
2. **Information density over whitespace** — compact layouts, clear hierarchy
3. **Predictable and deliberate** — minimal animation, instant feedback
4. **Theme-native** — use semantic tokens, never hardcode colors. All three themes are first-class
5. **Accessible by default** — WCAG AA, keyboard nav, reduced-motion support

See `.impeccable.md` for full design context and technical notes.
